# Registro de Atención Veterinaria

Este registro de atención veterinario, es un proyecto solicitado para ser presentado en la evaluación n°2 del ramo de Frond-End. Se aplico responsividad par adaptación de plataforma utilizada (PC, Tablet, Smartphone).

## Tecnologías

HTML5
CSS
JavaScript

![Pagina principal](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20154701.png)

## Modo de uso

### Nuevo ingreso

Registra el nombre de la mascota
Especifica la especie
Nombre del propietario
Edad de la mascota

![Ingreso](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20154415.png)

las metricas se actializan al momento de ingresar los pacientes
![Metricas](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20154425.png)

los pacientes ingresados quedan el Sala Espera
![Sala-Espera](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20154515.png)

se puede filtrar los pacientes por nombre del paciente por Pendientes o Atendidos, y si es necesario por abecedario.
![fitro_nombre](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20154437.png)
![filtro_pendite](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20154452.png)
![filtro_A-Z](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20154515.png)

en el caso que no se registre alguno de los datos la casilla que falta por completar quedara en rojo y aparecerá un cuadro con la leyenda "Revisa los siguientes datos antes de continuar: \* y menciona la casilla pendiente "
![error_dom](../eva2/assets/img/Captura%20de%20pantalla%202026-07-09%20161020.png)
